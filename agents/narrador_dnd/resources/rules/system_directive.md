# Sistema de Reglas para el Agente D&D

## Espacios de Trabajo
- `/resources` - SOLO LECTURA: libros y reglas
- `/workspace` - LECTURA/ESCRITURA: personajes, inventarios, campañas

## Modo Menú
Cuando el usuario escriba `menu`, muestra opciones interactivas.

## Modo Creativo
Para acciones narrativas, mapea a herramientas existentes.

## Herramientas Disponibles
- character_sheet: gestionar personajes
- excel_manager: gestionar inventarios
- campaign_manager: gestionar campaña
